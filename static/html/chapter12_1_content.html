📊 Summary & Key Insights

Key Formulas Discovered
Using arrangement number (x):

Red Squares: y = 4x + 2

Black Squares: z = x² + 2

Using side length (x):

Red Squares: y = 4x - 6

Black Squares: y = x² - 4x + 6

Pattern Analysis

Red squares: Linear relationship - increases by 4 each time

Black squares: Quadratic relationship - increases by odd numbers (3, 5, 7, 9...)

Key insight: Same pattern can be expressed with different variables!

Formula Verification

Multiple correct formulas:

• y = 4x + 2

• y = 2(2x + 1)

• y = 4(x - 2) + 2 + 8

All represent the same relationship!
