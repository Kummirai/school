13.1 Describing and Doing Computations

Algebraic expressions are a way to describe computations using symbols and numbers. They help us represent rules and relationships efficiently.

Variables and Constants

Variable: A quantity that can change or take on different values. It is usually represented by a letter symbol (e.g., x, n, l, b, w).
Constant: A quantity that remains the same or has a fixed value. It is usually a number.
Example: Circles in Arrangements (from Activity 1)

In diagrams where circles are arranged in two rows:

  • The number of rows (2) remains the same. This is a constant.
  • The number of circles per row (n) changes for each diagram. This is a variable.
  • The rule to calculate the total number of circles is 2 × n. Here, 2 is a constant, and n is a variable.

Flow Diagrams

Flow diagrams illustrate a sequence of operations performed on an input number to produce an output number.

Example: Simple Flow Diagram

Consider the flow diagram: Input → × 2 → Output

If Input is 11, Output is 11 × 2 = 22.
If Input is 21, Output is 21 × 2 = 42.
If Input is 41, Output is 41 × 2 = 82.
Example: Complex Flow Diagram (from Activity 4)

Consider the flow diagram: Input → + 4 → × 5 → Output

To calculate the output:
  1. First, add 4 to the input number.
  2. Then, multiply the result by 5.
If Input is 10: (10 + 4) × 5 = 14 × 5 = 70.

Algebraic Expressions

An algebraic expression is a mathematical phrase that can contain numbers, variables, and operation symbols (like +, -, ×, ÷). It's an abbreviation for a computational instruction.

Example: From Words to Expressions (from Activity 8)
  • "Multiply by 4 and then subtract 8": 4x - 8
  • "Subtract 8 and then multiply by 4": 4(x - 8) (Notice the brackets for order of operations!)
  • "Add 15 and then divide by 5": (x + 15) / 5
  • "Divide by 5 and then add 15": x / 5 + 15

Important: Order of Operations (BODMAS/PEMDAS) is crucial when writing and interpreting expressions. Brackets ensure operations are performed in the intended order.

Example: From Expressions to Words (from Activity 9)
  • x × 4 + 7: "Multiply by 4, then add 7."
  • (x + 7) × 4: "Add 7, then multiply by 4."

13.2 Relationships Represented in Formulae

A formula is a special type of algebraic expression that shows a relationship between different quantities, often used to calculate a specific value.

Variables and Constants in Formulae

Example: Perimeter Formula (from Activity 13.2.1)

The formula for the perimeter of a rectangle is P = 2 × l + 2 × b or P = 2 × (l + b).

  • Variables: P (Perimeter), l (Length), b (Breadth) - these values can change for different rectangles.
  • Constant: 2 - this number always remains the same in the formula. It represents the two pairs of sides in a rectangle.

Solving Equations

When an algebraic expression is set equal to a value, it becomes an equation. In an equation, the letter symbol represents an unknown value that we need to find.

Example: Sindi's Father's Age (from Activity 13.2.2)

Formula: F = x + 37, where x is Sindi's age and F is her father's age.

If her father passed away when Sindi was 43 years old, we can find his age by substituting x = 43 into the formula:

F = 43 + 37

F = 80

So, her father was 80 years old.

Example: Jacob's Cell Phone Savings (from Activity 13.2.3)

Formula: y = 45 + 5 × w, where y is total savings and w is weeks.

  • Constants: 45 (initial savings), 5 (savings per week).
  • Variables: y (amount saved), w (number of weeks).

If Jacob wants a phone that costs R90, we can find out if he has enough by the eighth week (w = 8):

y = 45 + 5 × 8

y = 45 + 40

y = 85

Since R85 is less than R90, he will *not* have enough money by the eighth week.
