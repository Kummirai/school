1. Understanding Functions and Relationships

Definition

Functions and relationships involve understanding the connection between input numbers and output numbers. The input undergoes a process (a rule) to produce the output.

Input

The starting number that goes into the relationship.

Rule

The set of operations (e.g., multiplying, dividing, adding, subtracting) performed on the input to get the output.

Output

The resulting number after the rule has been applied to the input.

Common Tasks
  • Use given inputs to calculate outputs
  • Use given outputs to calculate inputs (working backward)
  • Use given inputs and outputs to determine the rule

2. Representing Relationships

Function relationships can be shown in different ways:

Flow Diagrams
  • Visually show the input, the rule (operations), and the output
  • Indicate what calculations are done and which output is connected to which input
Input → [Operation 1] → [Operation 2] → Output
Tables of Values
  • Display pairs of input and corresponding output numbers
  • Sometimes numbers might be missing, requiring you to determine the rule first
Input (x) Output (y)
1 4
2 7
3 ?

3. Determining the Rule

A common task is to find the mathematical rule that connects inputs to outputs:

Arithmetic Patterns (Constant Difference)
  • For many patterns, you can identify a constant difference between consecutive output numbers
  • This constant difference is usually the multiplier in your rule
Steps to Find the Rule
  1. Look at the outputs and find the difference between consecutive terms
  2. Take this constant difference as the number you multiply the input (x) by
  3. Test this partial rule with an input number
  4. If needed, add or subtract a constant to make it match
  5. Verify the complete rule with other input-output pairs
Working with Negative Numbers: When dealing with negative inputs or outputs, follow the same method. Pay close attention to signs when calculating differences and applying the rule.

4. Working Backwards (Inverse Operations)

When you have the output and need to find the input, you must reverse the operations:

Opposite Operations
  • Addition (+) ↔ Subtraction (-)
  • Multiplication (×) ↔ Division (÷)
  • Squaring (x²) ↔ Square Root (√)
Reverse Order

You must undo the operations in the reverse order they were applied. Think of it like unwrapping a present.

Example: If the rule is "multiply by 2 and then add 3", to find the input from an output:

  1. First subtract 3
  2. Then divide by 2

5. Order of Operations (BODMAS/PEMDAS)

Understanding the correct order of operations is essential for accurate calculations:

BODMAS stands for:
  • Brackets
  • Orders (Exponents)
  • Division and Multiplication (same level)
  • Addition and Subtraction (same level)
Key Rules
  • Operations within brackets are always done first
  • Exponents are done next
  • For operations on the same level, work from left to right
  • Order matters when mixing operations from different levels
  • If you are only multiplying and dividing, the order doesn't change the answer

6. Measuring vs. Counting

Differentiate between quantities that are measured and those that are counted:

Counting
  • Involves using whole numbers
  • Applies to discrete items (e.g., people, cars)
  • Cannot have fractions ("57.5 people")
Measuring
  • Involves quantities that can have decimals
  • Applies to continuous quantities (e.g., distance, temperature)
  • Context dictates whether decimals are acceptable

7. Squaring vs. Square Rooting

These are two important inverse operations:

Squaring
  • Means multiplying a number by itself
  • Example: "4 squared" is 4 × 4 = 16
  • Used when calculating the area of a square
Square Rooting
  • The opposite operation of squaring
  • Finds a number that, when squared, gives the original number
  • Example: √144 = 12 (because 12 × 12 = 144)
  • Used to find a square's side length from its area
Language Matters: Use the correct terms (squaring, square rooting) to describe these operations, as they are distinct mathematical concepts.

Tips for Solving Problems

Practice

The more you practice, the easier these concepts will become.

Show Your Work

Especially with decimals or multiple operations, showing steps helps prevent errors.

Test Your Rule

Always substitute values back into your determined rule to verify correctness.

Ask for Clarification

If a query is ambiguous, it's always best to ask for clarification.
