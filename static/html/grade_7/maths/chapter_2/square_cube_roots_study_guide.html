2.4 The Square Root and the Cube Root

This section explores the inverse operations of squaring and cubing a number: finding the square root and the cube root.

The Square Root

[cite_start]

The inverse to finding the square of a number is to find its square root[cite: 70]. [cite_start]The question "What is the square root of 25?" is the same as asking, "What number, when squared, equals 25?"[cite: 71].

[cite_start]

The answer is 5, because \(5 \times 5 = 25\)[cite: 72].

[cite_start]

The symbol used to indicate the square root of a number is \(\sqrt{}\)[cite: 87, 90]. [cite_start]For example, \(\sqrt{25} = 5\)[cite: 87].

Examples:

    [cite_start]
  • \(\sqrt{9} = 3\) because \(3 \times 3 = 9\)[cite: 72].
    • [cite_start]
  • \(\sqrt{49} = 7\) because \(7 \times 7 = 49\)[cite: 73].
    • [cite_start]
  • \(\sqrt{121} = 11\) because \(11 \times 11 = 121\)[cite: 74].

The Cube Root

[cite_start]

The inverse operation to finding the cube of a number is to find its cube root[cite: 76]. [cite_start]The question "What number, when cubed, equals 125?" is the same as asking, "What is the cube root of 125?"[cite: 77].

[cite_start]

The answer is 5, because \(5 \times 5 \times 5 = 125\)[cite: 78].

[cite_start]

The symbol used to indicate the cube root of a number is \(\sqrt[3]{}\)[cite: 88, 91]. [cite_start]For example, \(\sqrt[3]{125} = 5\)[cite: 88]. [cite_start]The '3' written outside the root symbol is essential to distinguish it from a square root[cite: 91].

Examples:

    [cite_start]
  • \(\sqrt[3]{27} = 3\) because \(3 \times 3 \times 3 = 27\)[cite: 79].
    • [cite_start]
  • \(\sqrt[3]{8} = 2\) because \(2 \times 2 \times 2 = 8\)[cite: 80].
    • [cite_start]
  • \(\sqrt[3]{343} = 7\) because \(7 \times 7 \times 7 = 343\)[cite: 79].
