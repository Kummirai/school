Grade 7 Mathematics: Chapter 3

Geometry of Straight Lines: Line Segments, Lines and Rays

3.1 Line segments, lines and rays

In geometry, it's important to understand the precise definitions of basic building blocks like line segments, lines, and rays. While we can draw representations of them, their true definitions describe their extent.

Line Segments

A line segment is a part of a line that has a definite starting point and a definite endpoint. Because it has clear boundaries, we can draw and measure line segments.

A line segment has a definite start and end.

Examples:

  • Each side of a quadrilateral (a four-sided shape) is a line segment.
  • Drawing a line 12 cm long means drawing a line segment.

Lines

A line is different from a line segment because it has no ends; it extends indefinitely in both directions. We cannot draw a complete line, so we represent it by drawing a line segment with arrows at both ends to show that it continues forever.

A line extends indefinitely in both directions.

Key characteristics:

  • Goes on indefinitely in both directions.
  • Cannot be measured.
  • Represented with arrows at both ends.

Rays (or Half-lines)

A ray (sometimes called a half-line) has a definite starting point but goes on indefinitely in only one direction. We draw its starting point and part of it, using an arrow at one end to indicate its infinite extension.

A ray has a starting point and extends indefinitely in one direction.

Key characteristics:

  • Has a definite starting point.
  • Extends indefinitely in one direction.
  • Represented with an arrow at one end.
